### 2. Bug 修复详情

#### 2.1 数据库初始化问题修复
- **问题：** 在 `backend/src/main/resources/schema.sql` 中，测试用户数据插入时未提供密码字段
- **影响：** 导致用户无法使用默认凭据登录
- **修复：** 为所有测试用户添加了默认密码（'123456' 的 BCrypt 哈希值）
- **修改文件：** `backend/src/main/resources/schema.sql`

#### 2.2 前端 Axios 配置语法错误修复
- **问题：** 在 `frontend/cock-vue3-vite/src/api/axios.ts` 中，响应拦截器缺少闭合花括号
- **影响：** 可能导致 JavaScript 语法错误
- **修复：** 修正了缺失的闭合花括号
- **修改文件：** `frontend/cock-vue3-vite/src/api/axios.ts`

#### 2.3 用户常量配置错误修复
- **问题：** 在 `frontend/cock-vue3-vite/src/constants/userConstants.ts` 中，多了一个多余的闭合花括号
- **影响：** 可能导致 JavaScript 语法错误
- **修复：** 移除了多余的闭合花括号
- **修改文件：** `frontend/cock-vue3-vite/src/constants/userConstants.ts`
